Performance-Based Clustering and Market Valuation

## 1. 연구 배경 및 문제의식

MLB 선수 연봉은 단순히 경기 성적만으로 결정되지 않는다.
연봉에는 시장 기대, 포지션 가치, 플레이 스타일, 희소성 등이 복합적으로 반영된다.

따라서 본 연구는 다음과 같은 질문에서 출발한다.

“이 선수는 정말 연봉만큼의 가치를 하고 있는가?”

그러나 이 질문은 선수 간 직접 비교로는 분석이 불가능하다.
서로 다른 플레이 스타일과 역할을 가진 선수를 동일 기준으로 평가하는 것은 구조적으로 불공정하기 때문이다.

이에 본 연구는 선수들을 플레이 스타일 기준으로 군집화(clustering) 한 뒤,
동일 유형 내에서 연봉 대비 성과를 비교하는 방식을 채택한다.

## 2. 데이터 개요 및 EDA

### 2.1 데이터 구성

기간: 2021–2025 MLB 시즌

대상:

타자 (Hitters)
투수 (Pitchers)

주요 변수:

성적 지표: WAR, wRC+, HR, PA, IP, SIERA 등
계약 지표: Average Annual Value (AAV)

데이터 출처:
MLB 세이버매트릭스 기록
US TODAY 연봉 데이터

### 2.2 탐색적 데이터 분석 (EDA) 주요 관찰

WAR 및 연봉(AAV)은 강한 오른쪽 꼬리 분포(skewness) 를 가짐
소수의 초고연봉 스타 플레이어가 전체 분산을 지배
동일 연봉대에서도 성과 편차가 매우 큼
투수와 타자는 완전히 다른 분포 특성을 보임 → 분리 분석 필요

- 정규성·등분산 가정이 성립하지 않음
→ 이후 분석에서 비모수 통계 기법 채택

## 5. ‘돈값’의 정량적 정의
### 5.1 개념적 정의

```
돈값 = 실제 성과 − 시장이 기대한 성과
```

### 5.2 기대 성과 (Expected Performance)

 - 동일 클러스터에 속한 선수들이 동일한 연봉(AAV) 수준에서 평균적으로 만들어내는 성과

 -> 클러스터 평균 WAR

### 5.3 실제 성과 (Actual Performance)

 - 해당 선수의 WAR

### 5.4 Value Index (핵심 지표)

```
Value Index = 선수 WAR − (클러스터 평균 WAR)
Value Index > 0 → 연봉 대비 성과 초과
Value Index < 0 → 연봉 대비 성과 미달
```

### 5.5 Salary Efficiency (보조 지표)

```
Salary Efficiency = WAR / AAV
```

클러스터 평균과 비교하여 상대적 효율 평가.

## 7. 통계적 검정 선택 근거
### 결정 트리 요약

연속형 종속변수 (WAR, Value Index)
        ↓
3개 이상 집단 (클러스터)
        ↓
정규성 위배 + 분산 불균형
        ↓
비모수 검정
        ↓
Kruskal–Wallis + Dunn post-hoc

### 최종 통계 설계

종속변수: Salary Efficiency, Value Index
독립변수: Player Cluster (4 groups)

검정:
Kruskal–Wallis test
Dunn test (Bonferroni / Holm)
