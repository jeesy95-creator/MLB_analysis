
# 야구 세이버매트릭스 & 클래식 스탯 정리 (분석·연구용)

==================================================
[1] 타자 (Hitter)
==================================================

## 1-1. 클래식 스탯 (Traditional)

| 지표 | 설명 |
|---|---|
| G | 경기 수 |
| PA | 타석 수 |
| AB | 타수 |
| H | 안타 |
| 1B / 2B / 3B | 단타 / 2루타 / 3루타 |
| HR | 홈런 |
| R | 득점 |
| RBI | 타점 |
| BB | 볼넷 |
| SO (K) | 삼진 |
| AVG | 타율 |
| OBP | 출루율 |
| SLG | 장타율 |
| OPS | OBP + SLG |
| SB / CS | 도루 / 도루 실패 |
| HBP | 몸에 맞는 공 |

## 1-2. 세이버매트릭스 (Advanced – 핵심)

### 공격 가치
| 지표 | 설명 |
|---|---|
| wOBA | 가중 출루율 (공격력 대표 지표) |
| wRC | 득점 창출 능력 |
| wRC+ | 리그·파크 보정 득점 (100=평균) |
| WAR | 대체선수 대비 종합 가치 |
| OPS+ | 파크 보정 OPS |

### 파워 / 타구 품질
| 지표 | 설명 |
|---|---|
| ISO | 순수 장타력 (SLG - AVG) |
| BABIP | 인플레이 타구 타율 |
| Hard-Hit% | 강한 타구 비율 |
| Barrel% | 이상적 타구 비율 |
| EV | 평균 타구 속도 |
| LA | 평균 발사각 |
| Max EV | 최대 타구 속도 |

### 선구안 / 접근
| 지표 | 설명 |
|---|---|
| BB% | 볼넷 비율 |
| K% | 삼진 비율 |
| O-Swing% | 존 밖 스윙 비율 |
| Z-Swing% | 존 안 스윙 비율 |
| Contact% | 컨택 성공률 |
| SwStr% | 헛스윙 비율 |

### 주루 / 부가 가치
| 지표 | 설명 |
|---|---|
| BsR | 주루 득점 기여 |
| UBR | 주루 종합 지표 |
| Spd | 스피드 지수 |
